# AI & ML Internship – Task 4

## Classification Models & Evaluation Metrics

This project demonstrates classification model building, evaluation metrics, ROC-AUC analysis, and handling imbalanced data.

## Dataset
Breast Cancer Dataset (scikit-learn)

## Tools Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Workflow
- Data Loading
- Train-Test Split (Stratified)
- Feature Scaling
- Logistic Regression Model
- Confusion Matrix & Classification Report
- ROC Curve & AUC Score
- Handling Class Imbalance
- Model Comparison

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Key Insights
- Accuracy alone is misleading
- Recall is important for medical problems
- F1-score balances performance
- Random Forest gives better generalization
